微积分1 ~ 微积分3 的内容还跟一般微甲班的内容差异不大，
基本上是按照 Stewart 课本的编排（详细单元范围可见 ceiba ），
从单变量函数的微分、积分，一路推到多变量函数的微分与积分。

但从 微积分4 开始讨论「多个不等式限制条件下的最适化问题」后，
就进到一个全新的境界了XD。


微积分1：

　极限、微分、一阶线性估计、均值定理、一阶微分作图
　不定型、L'hopital's rule、微分应用（最适化问题）


微积分2：

　积分、微积分基本定理、积分技巧、不定积分
　微分方程、参数式、极座标


微积分3：

　二次曲面、极限、偏微分、方向导数、拉格朗日乘值法 (重要！微4的基础！)
　重积分、泰勒级数


微积分4：

　- 线性代数的基本观念：
　　  　向量空间、矩阵、特征值和特征向量、可对角化矩阵、正交可对角化矩阵
　　  　二次型与判断矩阵的正定、负定性（Sylvester's criterion）
　　
　- 等式与不等式限制条件下的最适化问题
　  ( NDCQ → Lagrangian's function → F.O.C )
　- 不等式限制条件下的最适化问题的特例：KKT formulaion
　- 拉格朗日乘数的意义 (包络定理的 special case)
　- 包络定理 (Envelope Theorem)
　- 最适化问题的 S.O.C （判别极值候选点是极大或极小）
　　
　- 最后一个礼拜补充的内容 （虽然期考没有考出来，但算是补齐微4的内容）：
　　　NDCQ 不满足下的 Lagrangian function （Fritz John Theorem）
　　　Implicit Function Theorem
　　　一些其他的 CQ (constraints qualification)

ρ 考题型式、作业方式

作业：

会从课本的当周授课范围后面例题抽几题出来，每一次大概 8~12 题左右，
从简单的计算题，到难一点的证明题都可能出现。
似乎是写完有交就有分？助教可能会稍微改一下，但不管错的多寡似乎都算满分。


学习单：

由于是经济系的班，老师会利用学习单的形式带入很多微积分在经济学上的应用。
（像是 MC-MR, CS-PS, Cobb-Doublas function, MRS, peak-load pricing...等等）
我个人觉得蛮有趣的，让你在学完数学工具后能顺便认识它是怎么被应用到经济学上的。

另外，同去年版上的评价，四个人写一张学习单真的没甚么意义...
通常都是一个人写好之后，另外三个人在上面署名。
看的出本意是要学生一同讨论后完成，但制度面有太大的漏洞。
也因此下学期之后就没有分组交学习单的制度了，每个人都要自己写一份上传。

　　
小考：

小考是由助教出的，题目从作业和学习单上出，
但个人觉得题目难度随着出题助教不同差异很大...不是很简单就是很难。
但因为期考会 cover 一次考差的小考分数，所以考爆也不要太担心，期考拚回来就好XD


期考：

微1、微2 的期考和微甲类似，题型差异不大。但会有一、两题类似学习单上的题目。
由于模组班是下午考，通常可以问当天早上考的微甲朋友他们考了甚么XD。
考前准备记得多刷微甲考古题，基本上那些题型的解题思路有了不会太困难。

微3 因为疫情的影响，所以改成 线上+纸本 分开施测。
同样也是要把纸本作业、webwork、学习单上面的题目好好练习。

微4 延续去年独立出题

      ρ 考题型式、作业方式

         期考题型：微1～3基本上跟同天考的微甲题目有7成重叠，
                         馀下3成是学习单里面的内容
                   微4的是整张独立出题，主要也还是跟学习单的题型相同

                   不确定下学年是否相同，毕竟是第一年开
                   推测会往微4这种独立出题的方式靠近

         小考：一门课两次，题型会从纸本作业、WebWork、学习单出
               准备起来不算太困难，但一次小考满分就是15分
               也就是小考少一分，期末就少一分，
               而且常常期末的等第都取决小考的一两分，所以每次小考都粉刺激><

         学习单：这堂课的卖点之一，主要会选一些与课堂进度相关
                 在经济及商业上的应用，例如成本最小化、边际产量、边际替代率
                 每一张都蛮有趣的，也会入期考题
                 但四人缴一张的规定实在蛮没意义的，因为有些人停休
                 有些实习课也不来，最后几张根本都单干
                 建议之后可以改成一人缴一张，反正依结果来看
                 一个人也是写的出来的（笑

         3.希望同学可以借由自己的程度及需求修课
           修完三堂就能达到毕业门槛，全修即视为微甲
